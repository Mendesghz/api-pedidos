# API de Pedidos

API REST desenvolvida com Node.js, Express e MongoDB.

## Tecnologias
- Node.js
- Express
- MongoDB

## Instalação

1. Instalar dependências
npm install

2. Rodar servidor
node server.js

Servidor roda em:
http://localhost:3000

## Rotas

POST /order -> criar pedido  
GET /order -> listar pedidos  
GET /order/:id -> buscar pedido  
PUT /order/:id -> atualizar pedido  
DELETE /order/:id -> deletar pedido
